### 關鍵架構洞見

**小天的設計哲學**：
- Claude Code（Coding LLM）是大腦 + 決策者
- `opc image`、`opc image analyze` 是工具（tool use）
- SKILL.md 和 references/image.md 是「教 Claude Code 怎麼用這些工具」的文件
- **沒有另外的 harness 執行檔**，Claude Code 本身就是 harness

小天的 flow：
```
用戶 → Claude Code → 自己生 JSON prompt
                   → 呼叫 opc image 生圖
                   → 呼叫 opc image analyze 看結果
                   → 自己思考改進 → 重複
```
